## 二、运行SASRec快速实验

### 1、模型训练

执行命令进行模型训练：

```bash
cd ~/research-training/lesson03/sasrec_smoke_demo
conda activate lesson03
CUDA_VISIBLE_DEVICES=0 python sasrec_demo.py \
  --device cuda \
  --epochs 10 \
  --batch-size 32 \
  --max-len 20 \
  --hidden-size 64 \
  --num-heads 2 \
  --num-blocks 2 \
  --allocator-limit-mib 700 \
  --gpu-budget-mib 1024
```
